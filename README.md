# 🚉 Railway Station Risk Dashboard

## 📌 Project Overview

This project implements an Unsupervised Machine Learning system 
to classify railway stations into different safety risk levels.

The system uses PCA for dimensionality reduction and 
K-Means clustering for grouping similar safety patterns.

An interactive web dashboard is built using Streamlit 
with secure login authentication.

---

## 🎯 Objective

To automatically detect abnormal safety patterns 
in railway station data without labeled target values.

---

## 🛠 Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Plotly
- Streamlit
- SQLite

---

## 🤖 Machine Learning Techniques

- Feature Scaling
- PCA (Principal Component Analysis)
- K-Means Clustering
- Distance-based Risk Scoring

---

## 🔐 Features

- Login & Signup Authentication
- Real-time Dataset Upload
- 3-Level Risk Classification (Low / Medium / High)
- Risk Score Calculation
- Interactive Dashboard Visualization
- Downloadable Risk Report

---
